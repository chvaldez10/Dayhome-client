# 🚀 Automate the CSV Extraction from the CRAFTS Website

## Setup 🛠️

1. Clone the repository 📂
2. Create a `.env` file in the root folder 📝
   ```bash
   CRAFTS_USERNAME=
   CRAFTS_PASSWORD=
   CRAFTS_LOGIN_URL=
   CRAFTS_ATTENDANCE_URL=
   EXPECTED_CRAFTS_CSV=
   ```
3. Run `npm install` 📦
4. Run `npx cypress open` to open the Cypress GUI 🖥️
